With hieWPwhois Plugin you can query domain status and whois (DomainavailibilityCHecK).

== Installation ==
1. Upload the directory `hieWPwhois´ with included files to the `/wp-content/plugins/´ directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Edit the config and activate the widget.
